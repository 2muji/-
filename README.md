# モノミッケ

落とし物を登録・検索し、見つかったものについてチャットで受け取り相談ができるシンプルな Web アプリです。

## 主な機能

- 落とし物の登録
  - 落とし物の名前
  - 特徴
  - 発見者
  - 発見場所
  - 発見時刻
  - 写真
- 複数条件をまとめた検索
  - 名前
  - 特徴
  - 発見者
  - 発見場所
  - 発見時刻
- 受け取り方法を相談するチャット
- ブラウザの `localStorage` への保存

## 使い方

1. `index.html` をブラウザで開きます。
2. 左側フォームから落とし物を登録します。
3. 右側の検索欄にキーワードを入れると絞り込みできます。
4. 「チャットで連絡する」から受け取りの相談ができます。

## 注意

この版は **フロントエンドだけで動くデモ / MVP** です。
そのため、データ保存とチャットは同じブラウザ内の `localStorage` を使っています。

本当に複数人で使えるアプリにするには、次の追加が必要です。

- ログイン機能
- サーバー側データベース
- リアルタイムチャット機能
- 画像アップロード用ストレージ
- 権限管理
